Listas - HTML

Lista ordenada é utilizada para criar marcações e organização dentro da nossa página.

Lista Ordenada

É um tipo de lista que pode ser organizada de forma numérica

Exemplo de Lista Ordenada

  1. Primeiro ítem
  2. Segundo ítem
  3. terceiro ítem

Lista Não Ordenada

É um tipo de lista que pode ser organizada por meio de marcadores

Exemplo de Lista Não Ordenada

  • Primeiro ítem
  • Segundo ítem
  • terceiro ítem
